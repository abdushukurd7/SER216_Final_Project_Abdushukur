# Digital Parking Permit System

##  Project Overview
The Digital Parking Permit System is a software solution designed for a university to manage vehicle registration, parking permit requests, and parking verification. The system replaces manual processes with a structured digital workflow, improving efficiency, accuracy, and security.

---

##  Features
- Vehicle registration for students and staff  
- Parking permit request system  
- Parking zone availability check  
- Permit approval process  
- Parking verification by security staff  
- Violation recording  

---

##  System Workflow
The system follows this process:

Vehicle Registration  → Permit Request  → Zone Check  → Permit Approval  → Parking Verification

---

##  Testing
The system was tested using:
- System Testing – verifies full system functionality  
- Acceptance Testing – ensures user requirements are met  
- Regression Testing – ensures existing features still work after updates  

---

##  Defect Analysis
A defect was identified where a valid parking permit was incorrectly shown as expired during verification.  
- Category: Logic Error  
- Severity: High  
- Priority: High  
- Cause: Incorrect date validation or system time mismatch
